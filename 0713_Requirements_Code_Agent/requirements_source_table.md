#### 1. 專案背景

```text
本專案為「校園外賣訂餐與取餐管理系統」，目的在整合學生訂餐、店家接單、校內配送及定點取餐流程，提升校園餐點配送效率。本次實作僅完成第一個可操作切片，不涉及正式部署、資料庫或第三方服務。
```
