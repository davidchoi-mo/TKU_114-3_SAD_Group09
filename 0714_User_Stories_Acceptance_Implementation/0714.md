### M. 更新後的程式碼代理實作任務書

一、專案名稱

校園外賣訂餐與取餐管理系統

二、目前系統狀態

目前已建立第一個可操作切片的基本原型，使用 HTML、CSS、JavaScript 與假資料實作。

既有原型包含：

1. 學生瀏覽合作店家與菜單。
2. 學生選擇餐點並送出訂單。
3. 訂單送出後顯示成功訊息。
4. 店家可在訂單列表查看學生送出的新訂單。
5. 店家可將訂單狀態更新為「已接單」或「製作中」。

目前仍需確認或修正：

1. 未選擇餐點時是否會錯誤建立訂單。
2. 連續按下送出按鈕是否會建立重複訂單。
3. 學生端與店家端顯示的訂單資料是否一致。
4. 訂單狀態是否依既定順序更新。
5. 狀態更新後，學生端是否顯示最新狀態。
6. 主要介面文字是否全部使用清楚的繁體中文。

三、本次目標

修正第一個可操作切片，使學生可以完成瀏覽菜單、選擇餐點及送出訂單，店家可以查看該筆訂單並依流程更新訂單狀態。

本次完成後，原型至少應通過下列核心驗收條件：

1. AC-01-01：學生可查看合作店家及可販售餐點。
2. AC-02-01：選擇餐點後可成功建立一筆訂單。
3. AC-02-02：未選擇餐點時不得建立訂單。
4. AC-02-03：連續送出時不得建立重複訂單。
5. AC-03-01：學生可查看訂單內容與目前狀態。
6. AC-04-01：店家可在訂單列表查看同一筆新訂單。
7. AC-05-01：店家可將新訂單更新為「已接單」。
8. AC-06-01：店家可將已接單訂單更新為「製作中」。
9. AC-06-02：不得跳過必要狀態直接更新為後續狀態。
10. AC-NFR-01-01：主要操作與狀態使用清楚的繁體中文。

四、保留項目

以下既有內容必須保留，不得刪除或破壞：

1. 合作店家與菜單瀏覽畫面。
2. 餐點名稱、價格及販售狀態顯示。
3. 學生餐點選擇與送出訂單操作。
4. 訂單建立成功訊息。
5. 店家訂單管理畫面。
6. 店家查看新訂單功能。
7. 使用 HTML、CSS、JavaScript 與假資料的既有技術方式。
8. 繁體中文介面。

五、修正項目

IMP-01：檢查菜單與介面文字

1. 顯示店家名稱、餐點名稱、價格及販售狀態。
2. 所有主要按鈕、提示訊息及訂單狀態使用繁體中文。
3. 不顯示與本次需求無關的功能。

IMP-02：修正訂單建立流程

1. 學生必須至少選擇一項可販售餐點，才能建立訂單。
2. 未選擇餐點時，顯示「請至少選擇一項餐點」。
3. 訂單成功建立時，只能產生一個訂單編號及一筆訂單。
4. 短時間內連續按下送出按鈕，不得建立重複訂單。
5. 訂單成功後顯示「訂單建立成功」。

IMP-03：修正學生端與店家端資料一致性

1. 學生送出訂單後，店家訂單列表立即新增同一筆訂單。
2. 學生端與店家端的訂單編號必須相同。
3. 雙方顯示的店家、餐點、數量、取餐點與目前狀態必須一致。
4. 學生端應能查看目前訂單狀態。

IMP-04：修正訂單狀態更新流程

1. 訂單建立後的初始狀態為「新訂單」。
2. 店家可將「新訂單」更新為「已接單」。
3. 店家可將「已接單」更新為「製作中」。
4. 店家可將「製作中」更新為「待配送」。
5. 不得跳過必要狀態直接更新為「已送達取餐點」。
6. 重複選擇目前狀態時，不得新增重複訂單或造成錯誤。
7. 店家更新狀態後，學生端必須顯示最新狀態。

六、新增項目

本次只允許新增下列必要項目：

1. 未選擇餐點時的繁體中文錯誤訊息。
2. 防止重複送出訂單的控制。
3. 學生端訂單狀態顯示區。
4. 店家端依順序更新狀態的操作控制。
5. 狀態更新後同步重新顯示訂單資料的邏輯。
6. 完成驗收所需的簡單假資料與測試情境。

七、刪除項目

如既有原型中出現下列未經需求確認的項目，應刪除或停用：

1. 正式登入或帳號註冊。
2. 線上付款或第三方金流按鈕。
3. GPS 定位、即時地圖或路線規劃。
4. Email、簡訊或手機推播設定。
5. 優惠券、會員點數、評分或促銷功能。
6. 管理員後台。
7. 歷史訂單與再次訂購功能。
8. 未在本任務書中列出的新角色或新流程。

八、本次不做

本次不得實作：

1. 正式登入及權限驗證。
2. 後端伺服器。
3. 正式資料庫。
4. 線上付款及第三方金流。
5. 即時 GPS 定位。
6. 配送路線最佳化。
7. 校務系統或第三方系統整合。
8. Email、簡訊及外部推播通知。
9. 店家拒單、學生取消訂單及退款流程。
10. 校內配送員的完整配送畫面。
11. 正式部署。

九、技術與資料限制

1. 前端技術：HTML、CSS、原生 JavaScript。
2. 後端技術：不使用。
3. 資料方式：使用程式內假資料（Mock Data）。
4. 外部套件：不使用。
